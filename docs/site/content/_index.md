title = "LLM Serving Pack"

The **nebari-llm-serving-pack** is a Kubernetes operator for serving large language models on a
Nebari cluster via llm-d, with per-model OIDC access control, API key management, and Envoy AI
Gateway integration.

Start with the [Quickstart](/quickstart/) for the fastest path to a running model, or the
[Installation](/installation/) runbook for a production ArgoCD/Nebari deployment.
